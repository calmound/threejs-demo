# 3D 城市可视化项目

这是一个基于 Three.js 的 3D 城市可视化项目，展示了一个动态生成的虚拟城市环境。

## 功能特性

- **3D 场景渲染**：使用 WebGL 技术渲染 3D 环境
- **动态建筑生成**：随机生成 100 个立方体建筑物
- **平滑动画**：使用 GSAP TweenMax 实现流畅的动画效果
- **自动相机控制**：相机自动环绕场景旋转
- **性能监控**：实时显示 FPS 和渲染性能
- **响应式设计**：支持窗口大小调整

## 技术栈

- **Three.js** - 3D 图形渲染库
- **GSAP TweenMax** - 动画库
- **Stats.js** - 性能监控
- **WebGL** - 硬件加速图形渲染

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 确保浏览器支持 WebGL
3. 观察动态生成的 3D 城市环境

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 注意事项

- 需要支持 WebGL 的现代浏览器
- 建议使用独立显卡以获得更好的性能
- 如果遇到性能问题，可以减少建筑物数量（修改 index.js 中的循环次数）
